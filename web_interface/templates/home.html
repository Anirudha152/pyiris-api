Generator

Generator Interface allows you to dynamically add and remove functionality to the deployable scout.
Scout can be compiled for functionality on all systems.
Supports source code encryption.

Listeners

Listeners allow you to manage network interfaces which are the actual programs handling and allowing connection to PyIris
They can connect out to bind TCP Shell scouts

Scouts

Scouts interface lets you manage and interact with scouts (the payloads which connect back to you)
Constantly expanding functionality to allow scouts to do so much more
Connecting and using scouts requires unique security key to match (the one above)
